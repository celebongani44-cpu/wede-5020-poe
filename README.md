## Project Overview

Jaguar Clothing is a newly established luxury streetwear brand positioned at the intersection of premium materials, limited drops, and bold urban styling. Because the brand is new, it currently has no website. This project delivers a clean, high-performance, brand-aligned website built from the ground up.

The website is designed to:

- Function as a **digital lookbook** that strengthens brand identity.
- Support **direct online sales** through clear product presentation.
- Provide a smooth **enquiry process** for customers (sizing, availability, bulk orders).
- Showcase **multiple physical points of presence** (pop-up/studio + warehouse/collection).

The website is being built using **HTML, CSS, and JavaScript**, version-controlled with **Git/GitHub**, and developed in **Visual Studio Code**.

## Website Goals and Objectives

Primary Goals

- Drive direct online sales while functioning as a high-end digital lookbook.
- Strengthen Jaguar Clothing’s luxury streetwear brand identity online.
- Establish a clean, scalable website architecture with no legacy constraints.

### Key Performance Indicators (KPIs)

- Monthly online sales revenue.
- Overall website conversion rate.
- Average time spent on product pages.
- Repeat customer purchase rate.
